Project Title:
PetAPITests (API test automation)

Tools:
REST Assured
Maven
TestNG
Allure

Requirements:
Java
Eclipse
Firefox

Usage:
1.Download the zip file or clone git repository.
2.Open Eclipse. File->Import->Existing Maven project->Navigate to the unzipped folder location-> Select (PetAPITests) project.
3.Navigate to src/test/java/APIAutomation/PetAPI.java file.
4.Right click & run as TestNG Test.

Reporting:
HTML report(open with firefox) can be found in test-output folder(\<eclipseworkspace>\PetAPITests\test-output\emailable-report).
Result JSON files(open with firefox) can be found in allure-results(\<eclipseworkspace>\PetAPITests\allure-results).
